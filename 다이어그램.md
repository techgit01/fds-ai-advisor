    end

    end

    end

    CARD --> DETECT
    AGI -->|음성 녹음 WAV| STT
    STT -->|텍스트| LLM
    LLM -->|판정 / 응답| TTS
    TTS -->|8kHz WAV| AGI

## 2. 상담 통화 흐름 (Sequence)

```mermaid
sequenceDiagram
    participant FDS as FDS 탐지
    participant AMI as caller.py (AMI)
    participant AST as Asterisk
    participant AGI as fds_agi.py
    participant AI  as Flask API
    participant CUS as 고객 핸드폰

    FDS->>AMI: HIGH 위험도 감지 → 발신 요청
    AMI->>AST: Originate SIP/iphone
    AST->>CUS: 전화 발신 🔔
    CUS->>AST: 수신 (통화 연결)
    AST->>AGI: AGI 스크립트 실행

    Note over AGI,AI: PHASE 1 — 본인확인
    AGI->>AI: POST /api/agi/tts (인사말)
    AI-->>AGI: 8kHz WAV
    AGI->>CUS: 음성 재생 "본인이 맞으신가요?"
    CUS->>AGI: 음성 응답 (녹음)
    AGI->>AI: POST /api/stt (음성 파일)
    AI-->>AGI: 텍스트 변환
    AGI->>AI: POST /api/agi/judge (phase=1)
    AI-->>AGI: verdict: confirmed / denied

    alt 본인확인 실패
        AGI->>AI: POST /api/agi/tts (작별 인사)
        AGI->>CUS: "실례했습니다"
        AGI->>AST: HANGUP → 통화실패
    else 본인확인 성공
        Note over AGI,AI: PHASE 2 — 거래확인
        AGI->>AI: POST /api/agi/tts (거래 질문)
        AI-->>AGI: 8kHz WAV
        AGI->>CUS: 음성 재생 "본인 거래 맞으신가요?"
        CUS->>AGI: 음성 응답 (녹음)
        AGI->>AI: POST /api/stt
        AI-->>AGI: 텍스트 변환
        AGI->>AI: POST /api/agi/judge (phase=2)
        AI-->>AGI: verdict: legitimate / fraud / uncertain
        AGI->>AI: POST /api/agi/tts (최종 안내)
        AGI->>CUS: 최종 결과 음성 재생
        AGI->>AST: HANGUP
    end
```

## 6. STT → LLM → TTS 데이터 흐름

```mermaid
flowchart LR
    MIC[🎙 고객 음성\nRTP 스트림]
    -->|Asterisk\nRECORD FILE| WAV_IN[WAV 파일\n8kHz mono]
    -->|POST /api/stt| WHISPER[Whisper\nlarge-v3]
    -->|한국어 텍스트| SANITIZE[입력 정제\n인젝션 방지]
    -->|POST /api/agi/judge| LLM[NIM LLM\ntemp=0]
    -->|verdict| JUDGE{판정}

    JUDGE -->|confirmed\nlegitimate| REPLY_GEN[응답 생성\ntemp=0.7]
    JUDGE -->|denied\nfraud| REPLY_GEN

    REPLY_GEN -->|POST /api/agi/tts| TTS[Supertonic\n44100→8kHz]
    -->|WAV 파일| PLAY[Asterisk\nSTREAM FILE]
    -->|🔊 고객 수신| SPEAKER[📱 고객 스피커]
```
